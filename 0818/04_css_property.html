01_css_property 종류 알아보기

css 언어는 '선택자 {속성:값;}' 으로 작성된다. 여기서 속성은 사용자가 선택한 요소에 다양한 서식을 적용하기 위한 종류에 해당한다.

css 속성의 종류

  • 글자 관련 : font-family(형태), font-size(크기), color(색), text_decoration(밑줄 유무,위치), text-transform(영문 대소문자 전환), font-style(기울게, 보통 전환), font-weight(굷게)
  • 문단 관련 : text-align(정렬), text-indent(들여쓰기), line-height(행간)
  • 크기 관련 : width(가로), height(높이)
  • 테두리 관련 : dorder(선)
  • 배경 관련 : background-color, background-image, background-size, background-position. . . .
  • 위치 관련 : position, float, transform:translate(가로, 세로)
  • layout 관련 : flex, grid, display
  • anmation 관련 : anmation, transition, @keyframes {내용}

1. 글자 관련 속성 사용하기 p.180

font-family : 글자 모양을 지정하고자 할 때 사용하는 속성이며, 한글글꼴을 지정하고자 할 떄는 '굴림체' 처럼 인용부호('')를 붙이며, 영문글꼴에서는 공백이 있는 fontname도 인용무호를('') 붙인다.

font-size : 글자 크기를 지정하는 속성이며, px, %, em, rem 단위가 있다. 기본값은 px 단위를 많이 사용하며, Browser에서 표시되는 기본값은 16px이다.

text-transform : 영문 - 소문자를 대문자로 , 대문자를 소문자로, 단어의 첫글자만 대문자로 변환한다.

글자 서식 정용시 단축 속성을 사용ㅎㄹ 수도 있다.
작성방법) font 기울기 두께 크기 / 줄높이 글꼴;
글자 크기와 글꼴은 반드시 작성해야 한다.
ex) font : bold 30px / 1.5 '궁서체';
