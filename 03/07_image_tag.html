07_Image_Tag

web 문서 안에서 사용되는 이미지 종류
[ .jpg(고화질), png(고화질, logo, icon, 투명배경이미지), gif(단순 컬러의 이미지나 움직이는 이미지, icon) ]

이러한 Image 종류들을 web 문서에 삽입하기 위한 tag element 가 img tag 이며, source의 약자인 'src' 속성과 image가 나오지 않을 때 대체할 문자인 'alt' 속성을 사용한다

[ 작성법 ] img src="삽입할 image의 address나 경로" alt="image가 나오지 않을 때 대체 할 문구"

1. image 삽입하기

절대주소를 사용하여 이미지 삽입하기
상대주소를 사용하여 이미지 삽입하기

URL - Uniform Resource Locator의 약자로서 WebServer에 올라간 File, 문서의 정확한 경로나 주소, 위치를 말함.

상대주소 : HTML 문서가 있는 곳이 기준이 되며, 하위에 img folder가 있을 경우 하위folder(./folderName/fileName.확장자)가 된다.

절대주소 : https://나 http://로 시작하여 'folderName/fileName.확장자'로 끝나는 주소를 말함.



2. image가 나오지 않을 때 대신해서 나오는 설명문구

alt : alternate의 약자로서 image가 나오지 않거나 응답이 없을 때 image를 대신해서 표시할 문구이다. 사용자가 혼란스럽지 않게 하기 위함이며, 시각 장애인이 image를 이해 하게 도와주는 언어이기도 하다. 음성낭독기를 사용하면 image의 alt속성에 저장된 문자를 읽어준다. Web접근성 규칙 중 하나이다.
