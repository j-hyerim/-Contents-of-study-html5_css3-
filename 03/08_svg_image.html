08_SVG_Img

svg : Scale Vector Graphic 의 약자로서 Vector 방식을 사용하여 image가 확대, 축소를 하여도 깨지지 않고 선명하게 나오는 image file 형식을 말한다.

W3C 에서 새롭게 만든 Image file format으로서 mobile, tablet 화면에서 화면 해상도에 따른 image를 깨끗하게 보여주기 위해서 사용하며 logo, icon 등에 활용된다.

svg Img 제작 : Vector 방식의 program인 'Illustrator"에서 제작하며, Desige 작업 후 'file> save as > svg' 확장자를 선택하여 image를 저장한다.
