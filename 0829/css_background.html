01_css_background(배경속성 p.183)

  • 1. background-color : 배경색상(#000000, rgb(), rgba(), hsl(), hsla())
  • 2. background-image : 배경이미지
  • 3. background-repeat : 배경반복설정
  • 4. background-position : 배경위치지정
  • 5. background-size : 배경크기지정
  • 6. background-origin : 배경이미지 기준위치 지정
  • 7. background-attachment : 배경이미지 고정
  • 8. 멀티백그라운드 : css3에서 추가된 속성으로서 여러 이미지를 한꺼번에 삽입할 수 있다.
색상코드
1. 색상명 : red, yellow, blue, black, white, green.......
2. hex코드 or 16진수 코드 : 0~9, a,b,c,d,e,f의 숫자와 문자를 조합하여 만든 코드이며 #00000(검정), #ffffff(흰색), #ff0000(빨강), #00ff00(초록), #0000ff(파랑)으로 작성된다.
3. rgb : red, green, blue 의 3가지 색상을 사용하여 색을 표현하는 코드로서 rgb(255,255,255)식으로 작성한다.
4. rgba : rgb색상코드에 alpha값을 사용하여 투명도를 적용할 수 있는 색상코드
5. hsl : hue(색조), saturation(채도), lightness(명도) 3가지 속성을 사용하여 색상을 표현하는 코드.
6. hsla : hsl색상코드에 alpha값을 사용하여 투명도를 적용하는 코드.
