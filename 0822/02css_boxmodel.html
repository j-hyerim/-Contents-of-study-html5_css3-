blouk 방식의 요소

p tag인 나는 blouk element입니다. 왼쪽에서 오른쪽까지 내용을 무조건 차지합니다. blouk element는 줄바꿈을 하지 않아도 자동으로 줄이 변경됩니다. 또한 가로크기, 세로크기 값을 설정하여 크기를 지정할 수 있습니다. blouk element의 종류로는 p, h1~h6, div, header, nav, main, section, article, aside, footer, address, dl, dt, dd, ul, li, ol, form, fieldset, gr, table 등이 있습니다.

위 문단요소의 성격이 변경되어 현 문장이 올라갑니다.

inline 방식의 요소

span tag인 나는 inline element 입니다. 글자가 있는 곳 까지만 영역을 차지합니다. blouk element와 다르게 자동으로 줄이 변경되지 않고 br Tag를 이용하여 줄바꿈을 합니다. inline element는 크기 값을 가질 수 없습니다. 대신 display 속성을 blouk 또는 inline-blouk으로 변경하면 가능합니다. 대표적인 Tag로는 img, b, strong, sup, sub, input, label, button, textarea ... 등이 있습니다. span Tag는 tag 자체의 의미는 없지만 글자를 묶어서 서식을 적용하고자 할 때 사용합니다. span Tag는 tag 자체의 의미는 없지만 글자를 묶어서 서식을 적용하고자 할 때 사용합니다. 2. box modle이란? block Element와 lnline element가 가질 수 있는 size, margin, padding, border의 값들을 사용하여 배치하는 개념이다.

사용 속성 - 크기 속성, 테두리, margin, padding
width, height (p.162) : 가로 세로 크기를 지정하며, px, %등의 단위를 사용한다.
border (p.167) : 테두리를 지정하고자 할 때 사용하며 축약법으로는 'border: 두께 모양 색상' 으로 작성한다.
margin (p.164) : 선택한 요소를 기준으로 바깥쪽으로 여백을 설정할 때 사용한다.
padding (p.165) : 선택한 요소를 기준으로 안쪽으로 여백을 설정할 때 사용한다.
