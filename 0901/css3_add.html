1. border-radius

사용자가 선택한 요소에 모서리 부분을 둥글게 만드는 속성이다

button, box의 모서리를 유연하게 처리하기 위해 사용하며, image에도 적용이 가능하다.

border-radius : px, % or border-radius : 0px 0px 0px 0px; (왼쪽 상단부터 시계방향)

3. shadow 속성

그림자 속성은 text, image, block element에 그림자 효과를 적용하고자 할 때 사용한다.

[ 작성법 ] text-shadow or box-shadow : x y offset color;

4. transform (2d,3d) -변형

transform 속성을 사용하여 사용자가 선택한 요소의 크기를 변경(scale), 기울기(skew), 회전(rotate), 위치이동(translate) 을 하기 위한 속성이다.

  • scale - 크기를 조정하기 위한 속성으로서 % 단위를 소수점으로 환산하여 입력한다.
  • skew - 기울기를 위한 속성으로서 각도 입력시 단위 'deg'를 입력한다.
  • rotate - 회전을 위한 속성으로서 각도 입력시 단위 'deg'를 입력한다.
  • translate - 위치를 지정하기 위한 속성으로서 position:relative 속성과 비슷하다.

5. transition - 전환

transition(전환) 속성은 사용자가 선택한 html element에 마우스를 올리거나 click 을 했을 때, 첫 장면에서 마지막 장면으로 자연스럽게 넘어가는 효과를 구현하고자 할때 사용한다.

mobile toggle menu를 click 할 때 부드럽게 나오는 animation, pc 화면에서 main navigation을 마우스 오버시 나오는 sub menu, main화면에 좌, 우 버튼 클릭 시 슬라이드 되는 이미지 등 다양한 곳에 활용이 가능하다.

9월 초특가 이벤트

CSS에서 사용된 속성들은 브라우저의 종류별, 버전별 호환성 여부를 확인하고 사용해야 한다. 사용자의 시스템, 브라우저 버전에 따라 서식이 적용 될 수도, 안 될 수도 있기때문에 앞으로 새로 추가되는 속성들은 호환성을 고려하여 브라우저 별로 속성앞에 '접두사'를 붙여서 임시로 적용해야한다.

접두사를 일일히 붙여주면 소스코드가 많아지고 개발자의 입장에서 귀찮은데, 이것을 자동으로 붙여주는 스크립트를 사용하면 더이상 신경을 쓰지 않아도 된다.

https://projects.verou.me/prefixfree/ 사이트를 방문하여 왼쪽 상단 동그란 메뉴를 마우스 오른쪽 메뉴를 클릭하여 js 파일을 다운로드 하고 현재 문서에 script src="" 로 연결한다.
