grid layout

가로축, 세로축 2차원 배열을 하기 위한 layout

grid 속성은 flex 속성과는 다르게 두 방향(가로, 세로)으로 layout을 만들기 위한 System이다.

grid는 부모요소(contaner)안에 자식요소를 배치하는데 자식요소를 보통 'grid item'이라고 한다.

flex 속성처럼 부모요소에 display:grid; 속성을 적용한다.

css 학습 참고 사이트 - https://www.w3schools.com/css/css_grid.asp

css grid 속성을 학습하기 위한 게임 사이트 - http://cssgridgarden.com/#ko
