css_keyframe_animation

animation keyframe 속성은 @keyframes 속성 안에서 animation 내용을 작성한 후 사용자가 지정한 요소에 animation 이름을 지정하여 자동으로 움직이게 한다.

        [ 작성법 ]
        @keyframes 애니메이션이름{
            0% {실행할 서식}
            100% {실행할 서식}
        }
  • animation-name : 적용할 animation 이름
  • animation-duration : animation 시간 설정
  • animation-iteration-count : animation 반복 횟수
  • animation-delay : animation 지연 시간 설정

1. 자동으로 회전하는 상자

2. 상하좌우로 움직이는 원

3. font awesome icon 을 활용한 menu navigation
