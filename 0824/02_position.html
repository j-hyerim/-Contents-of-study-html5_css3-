position 속성이란? layout 작업 시 사용자가 부모 요소 Box 안에서 원하는 content를 선택하여 배치하기 위한 속성으로서 static(요소가 가지고 있는 기본값), absolute(절대배치), relative(상대배치), fixed(고정배치), inherit(상속)의 종류가 있다.

position 속성 사용시 위치를 지정하기 위해서는 top, bottom, right, left 속성을 사용한다.

Position 속성을 사용하면 float 속성처럼 content와 content끼리 겹침 현상이 나타나는데, 위, 아래 순서를 변경하고자 할 때는 z-index 속성을 사용한다.

z-index 속성은 먼저 작성된 Tag의 z-index값을 0으로 주며, 나중에 작성된 Tag는 순서대로 자동 번호가 매겨진다. z-index 속성을 사용하여 위, 아래 순서도 변경이 가능하다.

1. absolute

2. relative

3. fixed

top
