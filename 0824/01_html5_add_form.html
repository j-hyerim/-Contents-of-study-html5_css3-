로그인

아이디찾기 | 비밀번호 찾기


placeholder : 사용자가 입력할 내용을 미리 알려주어 오류가 나지 않도록 방지해줌. 웹접근성 측면에서 도움이 많이 됨.

2. search : 사용자가 입력 양식에 키워드를 입력하여 검색할 수 있도록 하고자 할 때 사용하는 양식으로서 단어를 입력하면 오른쪽 끝에 'X'표시가 뜨며 클릭시 내용이 삭제됨.

3. email : 이메일 주소를 입력하기 위한 박스로서 입력양식이 맞지 않을 경우 에러가 난다.

4. url : 사용자가 도메인 주소를 입력할 수 있도록 하는 양식이며, 주소형식이 맞지 않을 경우 에러메세지가 나온다.

5. tel : 전화번호를 입력하기 위한 양식(모바일에서 키패드가 뜬다)

6. number : 숫자를 입력하기 위한 양식으로 직접입력하거나 버튼을 사용하여 설정

7. range : 숫자범위를 슬라이드바로 조정

8. date : 년, 월, 일을 선택하기 위한 요소로서 클릭시 달력 창이 활성화됨.(예약, 예매)

9. datetime : 연, 월, 일, 시, 분까지 선택하기 위한 요소로서 html5에서 없어졌다가 5.1부터 다시 사용

종료날짜 :

10. month : 년도와 월을 선택하기 위한 양식

11. week : 년도와 주를 선택하기 위한 양식

12. time : 시, 분을 선택하기 위한 양식

13. color : 색상을 선택하기 위한 양식

14. progress : 진행정도를 나타내는 바를 만들기 위한 태그로서 max, value 속성을 사용
