image를 대체하기 위한 icon font 기술

web font를 사용하여 특수문자(자음+한영키)를 삽입하듯이 다양한 아이콘을 삽입할 수 잇는 기술을 말하며, icon font를 제공하는 업체로는 폰트어썸, 부트스트랩, 구글, XEICON 등이 있다.

icon font는 기존의 icon을 image로 삽입하는 sprite image 방식의 단점을 보완한 아이콘 삽입방법으로서 다양한 디바이스에서 활용이 가능하고 css web font 를 사용하여 서식을 글자처럼 꾸밀 수 있는 장점이 있다.

w3schools Site의 icon 카테고리에서 사용법 확인

1. 폰트어썸을 사용하여 icon 삽입하기

폰트어썸 cdn 주소를 작성하고 ,span ,i Tag와 icon의 class명을 찾아서 작성하면 icon을 표시할 수 있다.

https://cdnjs.com/libraries/font-awesome

2. 부트스트랩을 사용하여 icon 삽입하기

bootstrap link 주소를 작성한 후, 삽입하고자 하는 icon의 class 명을 i, span Tag에 작성하면 icon이 표시된다.

3. 구글 웹폰트를 사용하여 icon 삽입하기

google font link주소를 작성한 후, class 명은 무조건 'material-icons'을 작성해야하며, 삽입하고자 하는 icon의 이름을 i, span tag 안에 작성한다.
