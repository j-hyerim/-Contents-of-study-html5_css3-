02_css_selector

css selector는 서식을 적용하기 위해 가장 먼저 접근해야하는 순서이고, 설정해야 하는 방법이며, 선택자의 종류로는 50여가지가 있다.

  • *class selector
  • *id selector
  • *tag selector
  • 전체 선택자
  • 자손, 후손 선택자
  • 속성 선택자
  • 동의 선택자
  • 속성 선택자
  • 구조 선택자(일반,형태구조)
  • *반응, link 선택자
  • 전,후문자 선택자
  • 부정 선택자
  • *class 결합, Tag와 class 결합
  • *Groub 선택자

1. class selector (.)

class selector는 사용자가 지정한 범위에 서식을 적용하기 위한 선택자로서 언제든 중복해서 서식을 적용할 수 있다. class selector는 class이름 앞에 '.' 기호를 넣어서 작성하는 것이 특징이고, WebPage 서식 작성시 많이 사용되는 선택자 중 하나이다.

2. id selector (#)

사용자가 지정한 범위에서 서식을 적용하기 위한 선택자로서 한번 적용된 서식은 재사용이 불가능한 선택자이다. 중복되지 않는 서식을 작성하기 위한 선택자이며, 주로 layout이나 javascript, jquery에서 사용자 동적 기능을 구현하고자 할 때 사용한다.

3. tag selector (tag)

html 문서내에 있는 사용자가 지정한 Tag 요소에 서식을 한꺼번에 적용하고자 할 때 사용하기 위한 선택자이며, 사용자가 일일히 찾아서 적용하지 않아도 되기 때문에 편하지만 주의해야 한다. (원하지 않는 부분가지 서식이 적용될 수 있기 때문)
