01 css 소개와 문법 p.135

css란? - Cascading style sheets 의 약자로서 WebPage의 서식을 적용하기 위한 언어이며, layout, 글자모양, 문단모양, 배경색, 배경Image, image크기 변경, Hyperlink 서식, 동적인 contents 등 다양한 서식을 작성하여 web Page를 표현할 수 있다.

Web 문서 안에서 동작되며 모든 Browser에서 WebSite가 동일하게 보여지는 효과를 얻을 수있으며, Tag와 서식이 분리가 되어 있어서 Site 유지보수가 간편하며, Design의 통일성을 보여줄 수 있다.

1. css 작성방법

        선택자 {
            속성:값;
            속성:값;
        }

Selector : 사용자가 서식을 적용하기 위하여 선택하는 대상을 말한다.

property : 서식의 종류로서 글자 관련, 문단 관련, 배경 관련, layout 관련 서식 등 다양하다.

Value : 속성을 사용하고자 할 떄 지정하는 내용이다. ex) color:red;, font-size:12px

2. css 주석처리

css에서는 /* 주석내용 */으로 작성하며 vs code program에서는 단축키 'ctrl + /'를 사용한다.
