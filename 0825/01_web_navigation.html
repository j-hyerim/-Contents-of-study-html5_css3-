01_Web_navigation

web page에서 사용자가 원하는 정보를 찾기 위한 가장 기본적인 수단으로서 navigation이라고 하며, 종류에는 main menu, sub menu, search menu, login menu, side menu, drop down mune, tep menu, 사용자의 현재 위치를 알려주는 'location' menu 등이 있다.

navigation 개발은 menu 목록을 나열하기 위해 목록Tag Element ul,li Tag를 사용하고, a Tag element로 hyperlink를 설정한다.

gnb : Global Navigation Bar의 약자로서 보통 main menu를 작성할 때 사용하는 명칭

inb : Local Navigation Bar의 약자로서 main menu 외에 sub menu나 content 영역에서 사용되는 menu를 작성하고자 할 때 사용하는 명칭이다.
